# heanbian-email

## 前提条件

JDK 17

## pom.xml

具体版本，可以从 Maven Central 获得

```
<dependency>
  <groupId>com.heanbian</groupId>
  <artifactId>heanbian-email</artifactId>
  <version>15.1</version>
</dependency>
```

## 使用示例

> EmailConfig

> EmailMessage

> EmailTemplate

说明：支持HTML邮件内容、附件File、接收人TO、抄送人CC、密送人BCC等。
